# Импорт книг из ЛитРес

WordPress плагин для импорта книг из каталога ЛитРес в формате CSV.

## Возможности

- Импорт книг из CSV файла ЛитРес
- Фильтрация по жанрам и авторам
- Батч-обработка для больших файлов (500 книг за раз)
- AJAX импорт с прогресс-баром
- Автоматическое создание таксономий (жанры, авторы)
- Поддержка UTF-8 BOM

## Требования

- WordPress 6.0+
- PHP 8.1+
- MySQL 5.7+

## Установка

1. Скопировать папку плагина в `wp-content/plugins/`
2. Активировать плагин в админке WordPress
3. Перейти в **Настройки → Постоянные ссылки** и нажать "Сохранить" (обязательно!)

## Использование

1. В админке WordPress: **Книги ЛитРес → Импорт**
2. Загрузить CSV файл или использовать `litresru.csv` из корня проекта
3. Нажать "Сканировать файл"
4. Выбрать жанры для импорта (обязательно)
5. Опционально выбрать авторов
6. Нажать "Запустить импорт"

## Структура CSV

Плагин ожидает CSV файл с разделителем `;` и следующими колонками:
- [0] - ID книги
- [1] - Название
- [3] - Жанр/категория
- [12] - Автор
- Остальные поля: цена, URL, описание, изображение

## Custom Post Type

Создаёт тип записи `zhsh_litres_book` с таксономиями:
- `zhsh_litres_genre` (жанры)
- `zhsh_litres_author` (авторы)

URL книг: `/book/название-книги/`

## Технические детали

- Батч-размер: 500 книг
- Лимит памяти: 2048M
- Timeout: 600 секунд
- Проверка дубликатов по `zhsh_litres_book_id`
